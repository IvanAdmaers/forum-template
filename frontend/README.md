# Frontend

Frontend part

## Development

1. Make sure .env.local file is fill

2. Run

```
npm run dev
```

## Deploy

1. Create an .env.local file and add variables from .env.example

2. Run 

```bash
npm run build
```

3. Run

```bash
npm run start
```

**If it is a first deploy NextJS application to server run:**
```bash
npx next telemetry disable
```
